# Nexus Graph Editor Shared Framework

This package contains the reusable graph editor implementation used by both NoDE-Lite and STAT.

NoDE-Lite and STAT now keep thin plugin wrappers while graph rendering, wire routing, inline subgraph expansion, drag/drop, view rules, table-node behavior, and serialization are maintained here.

NoDE-Lite launches with `default_view`. STAT launches with `stat.sequence_test_authoring`.
